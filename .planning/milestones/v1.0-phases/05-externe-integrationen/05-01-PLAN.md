<task type="auto">
  <name>Task 1: Stripe Client + Checkout Session API + Webhook API</name>
  <files>src/lib/stripe.ts, src/app/api/stripe/checkout/route.ts, src/app/api/stripe/webhook/route.ts, .env.example</files>
  <action>
1. Install stripe package: `npm install stripe`

2. Create `src/lib/stripe.ts` — Stripe client singleton:
   - `import Stripe from 'stripe'`
   - Export `stripe` instance created with `process.env.STRIPE_SECRET_KEY!`
   - Do NOT set apiVersion (use SDK default)
   - Set `typescript: true`
   - Also export a helper `createCheckoutSession(anfrageId: string, anfrageNummer: string, gesamtpreis: number, produktAnzahl: number)` that:
     - Creates a Stripe Checkout Session with mode 'payment', currency 'eur'
     - Uses `automatic_payment_methods` (no manual payment_method_types — per user decision: Stripe chooses based on country/amount)
     - line_items: one item with price_data (unit_amount = Math.round(gesamtpreis * 100)), product_data name = "Anfrage {nummer}", description = "{n} Produkt(e)"
     - metadata: { anfrage_id: anfrageId, anfrage_nummer: anfrageNummer }
     - success_url: `${process.env.NEXT_PUBLIC_SERVER_URL}/kunden/dashboard?payment=success`
     - cancel_url: `${process.env.NEXT_PUBLIC_SERVER_URL}/kunden/dashboard?payment=cancelled`
     - Returns the session object

3. Create `src/app/api/stripe/checkout/route.ts` — POST endpoint:
   - Parse JSON body with Zod: `{ anfrage_id: z.string().uuid() }`
   - Get Payload instance via `getPayload({ config })`
   - findByID the Anfrage, verify status === 'bestaetigt', otherwise 400
   - Call createCheckoutSession helper
   - Return `{ url: session.url }` with 200
   - On error: return 500 with error message

4. Create `src/app/api/stripe/webhook/route.ts` — POST endpoint:
   - CRITICAL: Use `await request.text()` for raw body (NOT request.json())
   - Get stripe-signature header
   - Verify with `stripe.webhooks.constructEvent(body, signature, process.env.STRIPE_WEBHOOK_SECRET!)`
   - On invalid signature: return 401
   - Handle event type 'checkout.session.completed':
     - Extract anfrage_id from session.metadata
     - Check session.payment_status === 'paid'
     - Get Payload instance, findByID the Anfrage
     - IDEMPOTENCY: If anfrage.status is already 'bezahlt', skip update, return 200
     - Otherwise update status to 'bezahlt' via payload.update()
   - Return `{ received: true }` with 200
   - Wrap entire handler in try/catch, log errors with console.error, return 500

5. Update `.env.example` (create if not exists) with:
   ```
   STRIPE_SECRET_KEY=sk_test_...
   STRIPE_WEBHOOK_SECRET=whsec_...
   ```
   Also add actual placeholder values to `.env` if it exists (check first).
  </action>
