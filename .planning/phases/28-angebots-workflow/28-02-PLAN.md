**On open/mount:** Fetch Settings Global to get default gueltigkeitTage and mwstSatz:
```typescript
useEffect(() => {
  if (!open) return;
  fetch("/api/globals/settings", { credentials: "include" })
    .then(res => res.json())
    .then(data => {
      const tage = data.angebots_gueltigkeit_tage || 30;
      const satz = data.mwst_satz || 19;
      setSettingsDefaults({ gueltigkeitTage: tage, mwstSatz: satz });
      setGueltigkeitTage(tage);
    })
    .catch(() => {});
  // Pre-fill price
  if (lastAngebotBrutto && (nextVersion || 1) > 1) {
    setBruttoCents(lastAngebotBrutto);
  } else {
    // Compute standard brutto from produkte
    import("@/lib/tax").then(({ calcGrossFromNet, calcTax }) => {
      const nettoSum = produkte.reduce((sum, p) => sum + (p.einzelpreis || 0) * (p.stueckzahl || 1), 0);
      const mwst = calcTax(nettoSum, settingsDefaults.mwstSatz);
      setBruttoCents(nettoSum + mwst);
    });
  }
}, [open]);
```

**Dual-price logic:**
- `calcNetFromGross(bruttoCents, mwstSatz)` gives nettoCents
- `mwstCents = bruttoCents - nettoCents` (derived, not separate calc)
- Display: "Netto: {formatCents(nettoCents)} | MwSt: {formatCents(mwstCents)}" below the brutto input
- On Einzelpreis change: recalculate bruttoCents as sum, set lastEditSource = "einzel"
- On Gesamtpreis manual change: set lastEditSource = "gesamt", value takes precedence
- "Letztes Edit gewinnt" is implemented via lastEditSource tracking

**Price-changed detection:**
```typescript
const standardNettoSum = produkte.reduce((sum, p) => sum + (p.einzelpreis || 0) * (p.stueckzahl || 1), 0);
const standardBrutto = calcGrossFromNet(standardNettoSum, settingsDefaults.mwstSatz);
const isPriceChanged = bruttoCents !== standardBrutto;
```

**Submit handler:**
```typescript
const handleSubmit = async () => {
  if (isPriceChanged && !begruendung.trim()) return;
  setSubmitting(true);
  setError(null);
  try {
    const res = await fetch("/api/angebot/erstellen", {
      method: "POST",
      headers: { "Content-Type": "application/json" },
      credentials: "include",
      body: JSON.stringify({
        anfrageId,
        bruttoCents,
        gueltigkeitTage,
        freitext: freitext.trim() || undefined,
        begruendung: isPriceChanged ? begruendung.trim() : undefined,
        einzelpreise: einzelpreise.length > 0 ? einzelpreise : undefined,
      }),
    });
    if (res.ok) {
      toast.success("Angebot erstellt und versendet");
      onSuccess();
      onClose();
    } else {
      const data = await res.json().catch(() => ({}));
      setError(data.error || "Angebot konnte nicht erstellt werden. Bitte versuchen Sie es erneut.");
    }
  } catch {
    setError("Angebot konnte nicht erstellt werden. Bitte versuchen Sie es erneut.");
  } finally {
    setSubmitting(false);
  }
};
```
