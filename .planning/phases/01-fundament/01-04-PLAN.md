**src/seed/utils.ts:**
- Helper function `findBySlug(payload, collection, slug)` that looks up an item by slug and returns its ID
- Helper function `clearCollection(payload, collection)` that deletes all items in a collection (for re-seeding)
- Helper function `clearAllCollections(payload)` that clears all collections in reverse dependency order

**src/seed/data/ files** (one per collection):
Each file exports an array of seed data objects. Use real Drutex data per user decisions:

**produkttypen.ts**: 2 items: Fenster (slug: fenster), Balkontuer (slug: balkontuer)

**materialien.ts**: 4 items: Kunststoff (slug: kunststoff, lieferzeit 4 weeks, garantie 5 years), Holz (slug: holz), Aluminium (slug: aluminium), Kunststoff-Aluminium (slug: kunststoff-aluminium). Only Kunststoff gets full data; others are created but sparse (per user decision).

**profile.ts**: 4 Kunststoff profiles per user decision:
- Iglo 5: standard, 70mm, 5 chambers, Uw 1.3, 2 seals, min 400x400mm, max 2600x2400mm
- Iglo Energy: premium, 82mm, 7 chambers, Uw 0.95, 3 seals, min 400x400mm, max 2600x2400mm
- Iglo Energy Classic: top, 82mm, 7 chambers, Uw 0.90, 3 seals, min 400x400mm, max 2600x2400mm
- Iglo Light: einstieg, 60mm, 5 chambers, Uw 1.4, 2 seals, min 400x400mm, max 2600x2400mm
- All reference material_slug: 'kunststoff' (resolved to ID during seeding)

**fluegelanzahl.ts**: 3 items: 1-fluegelig, 2-fluegelig, 3-fluegelig. 1-fluegelig for fenster+balkontuer, 2-fluegelig for fenster+balkontuer, 3-fluegelig for fenster only.

**oeffnungsarten.ts**: 4 items: Fest (fest, fuer_fenster: true, fuer_balkontuer: false), Kipp (kipp, both true), Dreh (dreh, both true), Dreh-Kipp (dreh-kipp, both true)

**fensterformen.ts**: At least 4 items: Rechteck (all fluegelanzahl, all oeffnungsarten), Rundbogen (1-2 fluegel, fest/dreh/dreh-kipp), Stichbogen (1-2 fluegel), Dreieck (1 fluegel, fest only). Reference fluegelanzahl and oeffnungsarten by slug (resolved during seeding).

**zusatzlichter.ts**: 2 items: Oberlicht, Unterlicht. Kombinierbar with 1-fluegelig and 2-fluegelig.

**farben.ts**: 10-15 colors per user decision:

- RAL-Sonderfarbe (~120 EUR surcharge): 1 placeholder entry "RAL-Sonderfarbe (auf Anfrage)"
- All colors: fuer_aussen: true, fuer_innen: true, erlaubte_materialien: [kunststoff]

**dichtungsfarben.ts**: 3-4 items: Schwarz, Grau, Weiss, Braun

**verglasungen.ts**: 4 items per user decision:
- 2-fach Standard (Ug 1.1, aufpreis 0)

- 3-fach Schallschutz (Ug 0.7, aufpreis 45/m2)

**schallschutz.ts**: 3 items: Schallschutzklasse 2 (33dB, 15 EUR), Klasse 3 (37dB, 30 EUR), Klasse 4 (42dB, 50 EUR)

**sicherheitsglas.ts**: 3 items: VSG Aussen (35 EUR), VSG Innen (35 EUR), VSG Beidseitig (60 EUR)

**sprossen.ts**: 3 items per user decision: Wiener Sprossen (40 EUR), Helima Sprossen (60 EUR), Aufgesetzte Sprossen (30 EUR)

**extras.ts**: 4-5 items: Standard-Griff (0 EUR, kategorie griffe), Sicherheits-Griff (25 EUR, griffe), Abschliessbarer Griff (35 EUR, griffe), Sicherheitsbeschlag (45 EUR, beschlaege), Pilzkopfverriegelung (30 EUR, beschlaege)

**preisregeln.ts**: 4 price rules for Kunststoff profiles:
