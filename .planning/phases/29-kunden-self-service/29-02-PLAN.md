Create new file following the annehmen/route.ts pattern but using FormData (not JSON) for file uploads:

```typescript
import { NextResponse } from "next/server";
import { z } from "zod";
import { MAX_FILE_SIZE, ALLOWED_FILE_TYPES, MAX_RUECKFRAGE_FILES } from "@/lib/upload-constants";

const antwortSchema = z.object({
  anfrageId: z.string().min(1),
  nachricht: z.string().min(10, "Mindestens 10 Zeichen erforderlich"),
});
```

POST handler logic:
1. `const formData = await request.formData()`
2. Extract `anfrageId` (string), `nachricht` (string), `dateien` (File[]) from formData
3. Validate text with Zod schema
4. Validate files: max 3 files (`MAX_RUECKFRAGE_FILES`), max 10MB each (`MAX_FILE_SIZE`), type in `ALLOWED_FILE_TYPES`. Return 400 with specific German error messages per UI-SPEC:
   - Too many files: "Maximal 3 Dateien erlaubt."
   - File too large: "Datei zu gross. Maximale Dateigroesse ist 10 MB."
   - Wrong type: "Nur Bilder (JPEG, PNG, WebP, HEIC) und PDF-Dateien erlaubt."
5. Rate limit: `checkRateLimit(`kunden-antwort:${ip}`, 5, 60_000)` — return 429 "Zu viele Anfragen. Bitte versuchen Sie es spaeter erneut."
6. Dynamic Payload import (same pattern as annehmen)
7. Load anfrage via `payload.findByID({ collection: "anfragen", id: anfrageId, depth: 0 })`
8. Validate `anfrage.status === "rueckfrage"` — return 400 "Antwort kann nur bei Status Rueckfrage gesendet werden."
9. Upload files to Media collection via Payload Local API:
```typescript
const mediaIds: string[] = [];
for (const datei of dateien) {
  const buffer = Buffer.from(await datei.arrayBuffer());
  const media = await payload.create({
    collection: "media",
    data: { alt: `Anhang zu Anfrage ${anfrage.anfrage_nummer || anfrageId}` },
    file: { data: buffer, mimetype: datei.type, name: datei.name, size: datei.size },
  });
  mediaIds.push(media.id);
}
```
10. Create StatusHistorie entry via Local API (bypasses access control):
```typescript
await payload.create({
  collection: "status_historie" as any,
  data: {
    anfrage: anfrageId,
    von_status: "rueckfrage",
    zu_status: "kundenantwort",
    zeitpunkt: new Date().toISOString(),
    kommentar: nachricht,
    geaendert_von: kontaktEmail, // or null for guest
    ...(mediaIds.length > 0 ? { anhaenge: mediaIds } : {}),
  },
});
```
Note: `geaendert_von` is a relationship to Users. For guest route, set it to undefined/null. For logged-in users, try to get user from request headers/cookies. Since StatusHistorie geaendert_von is optional and a relationship, we cannot store email directly. Instead, include the email in the kommentar: prefix with "Kundenantwort von [email]: ".

11. Update anfrage status:
```typescript
await payload.update({
  collection: "anfragen",
  id: anfrageId,
  data: { status: "kundenantwort" } as any,
});
```
12. Queue email event:
```typescript
const { queueEmailEvent } = await import("@/lib/email/queue");
await queueEmailEvent({
  eventType: "kundenantwort",
  anfrage_id: anfrageId,
  anfrage_nummer: anfrage.anfrage_nummer || "",
  status: "kundenantwort",
  kunden_email: kontaktEmail,
  kunden_name: kontaktName,
});
```
13. Return `NextResponse.json({ success: true }, { status: 200 })`

import { notFound } from "next/navigation";
import { getPayload } from "payload";
import config from "@payload-config";
import { RueckfrageFormular } from "@/components/kunden/rueckfrage-formular";
import type { Metadata } from "next";

export const metadata: Metadata = {
  title: "Rueckfrage beantworten | Christ Fensterhandel",

export default async function RueckfrageGuestPage({
  params,
}: {
  params: Promise<{ anfrageId: string }>;

  const { anfrageId } = await params;
  const payload = await getPayload({ config });

  let anfrage: any;
  try {
    anfrage = await payload.findByID({
      collection: "anfragen",
      id: anfrageId,
      depth: 1,
    });
  } catch {
    notFound();
  }

  if (!anfrage) notFound();

  // Must be in rueckfrage status
  if (anfrage.status !== "rueckfrage") {

          <p className="text-sm text-muted-foreground">
            Diese Anfrage erwartet derzeit keine Antwort.
          </p>
        </div>
      </main>
    );
  }

  // Find latest rueckfrage message from StatusHistorie
  const historieResult = await payload.find({
    collection: "status_historie" as any,
    where: {
      anfrage: { equals: anfrageId },
      zu_status: { equals: "rueckfrage" },
    },
    sort: "-zeitpunkt",
    limit: 1,
  });
  const lastRueckfrage = historieResult.docs[0] as any;
  const adminMessage = lastRueckfrage?.kommentar || "";

      <div className="rounded-xl border border-border bg-card p-6 shadow-sm">
        <h1 className="text-2xl font-bold text-foreground">
          Rueckfrage beantworten
        </h1>
        <p className="mt-1 text-sm text-muted-foreground">
          Ihre Anfrage {anfrageNummer} wartet auf Ihre Antwort zur folgenden Frage:
        </p>
        {adminMessage && (
          <div className="mt-4 rounded-lg border border-orange-200 bg-orange-50 p-4">
            <p className="text-sm text-orange-800">{adminMessage}</p>
          </div>
        )}
        <div className="mt-6">
          <RueckfrageFormular anfrageId={anfrageId} isGuest={true} />
        </div>
      </div>
    </main>
  );
}
