Create `src/components/admin/undo-redo-stack.ts` as a pure TypeScript class with NO React imports and only a type import from 'payload'. The class manages an undo/redo stack with these specifications:

**Class: UndoRedoStack**
- `constructor(maxSize: number = 50)` -- sets this.maxSize
- `push(state: FormState): void` -- Adds snapshot at currentIndex+1. If currentIndex is not at the end (user did undo then edited), discard everything after currentIndex first. If at maxSize capacity, shift the oldest snapshot out and decrement both currentIndex and floorIndex (floorIndex clamped to minimum -1).
- `undo(): FormState | null` -- If currentIndex <= floorIndex + 1, return null (at floor or below). Otherwise decrement currentIndex and return snapshots[currentIndex].
- `redo(): FormState | null` -- If currentIndex >= snapshots.length - 1, return null. Otherwise increment currentIndex and return snapshots[currentIndex].
- `markSaved(): void` -- Set floorIndex = currentIndex. This means undo cannot go below this point.
- `reset(): void` -- Set snapshots=[], currentIndex=-1, floorIndex=-1.
- `get canUndo(): boolean` -- currentIndex > floorIndex + 1
- `get canRedo(): boolean` -- currentIndex < snapshots.length - 1
- `get size(): number` -- snapshots.length

**Function: getDocKey(collectionSlug: string, id: string): string**
- Returns `${collectionSlug}:${id}`

**Function: createCleanSnapshot(formState: FormState): FormState**
- Iterates over Object.entries(formState)
- For each field, destructures `{ customComponents, ...rest }` to strip customComponents
- If `rest.rows` exists, map each row to strip its customComponents too: `{ customComponents: rowCC, ...rowRest }`
- Assigns cleaned field to `clean[path]`
- Returns `structuredClone(clean)`

Write tests FIRST (RED phase). Use plain objects as mock FormState -- each mock is a `Record<string, { value: any, valid?: boolean, customComponents?: object, rows?: Array<{ customComponents?: object, id?: string }> }>`. Run tests, confirm they fail, then implement the class, confirm GREEN.
