    Export `getFilteredOptions(step: number, cmsData: CMSData, selections: KonfiguratorSelections)` that returns filtered options for each step based on CMS relationship data:
    - Step 2: Filter materialien by produkttyp.erlaubte_materialien
    - Step 3: Filter profile by material.erlaubte_profile
    - Step 4: Filter fluegelanzahl by fuer_produkttypen containing selected produkttyp
    - Step 5: Filter oeffnungsarten by fuer_fenster/fuer_balkontuer based on produkttyp slug
    - Step 6: Filter fensterformen by erlaubte_fluegelanzahl and erlaubte_oeffnungsarten
    - Step 7: Return masse constraints from selected profil
    - Step 8: Filter farben by erlaubte_materialien + fuer_aussen/fuer_innen
    - Steps 1, 9, 10: Return all (no filtering)

    Export `getStepSchema(step: number, context?: { profil?: Profile })`.

Add aufpreis from verglasung, farben, schallschutz, sicherheitsglas, glasdekor, sprossen, extras. Return rounded to 2 decimals. Return 0 if no matching rule or missing data.
